F ixMyNotes.com

Make notes more readable...

Some courses give pdf notes with 4 slides in each page, this webapp uses Computer Vision to find the slides and then output a pdf with a single slide per page.

For now the program only works with symmetrically distributed slides. If slides do not maintain similar distance ratios to the document edges and/or other slides, the app might now work. On the other hand, slides can be anywhere as long as they maintain the mentioned ratios. This project is still in development! I Hope this helps someone's studying.
